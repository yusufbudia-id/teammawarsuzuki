# Cara Mengupload Brosur Produk

Folder ini berisi file brosur PDF untuk setiap produk.

## Struktur File

Setiap produk memiliki file brosur sendiri:
- `carry.pdf` - Brosur New Carry Pick Up
- `s-presso.pdf` - Brosur S-Presso
- `ertiga.pdf` - Brosur Ertiga
- `fronx.pdf` - Brosur Fronx
- `xl7.pdf` - Brosur XL7
- `grand-vitara.pdf` - Brosur Grand Vitara
- `jimny.pdf` - Brosur Jimny

## Cara Mengupload Brosur Asli

1. Siapkan file PDF brosur yang ingin diupload
2. Pastikan nama file sesuai dengan nama di atas
3. Timpa (replace) file dummy yang ada dengan file brosur asli Anda

**Contoh:**
Jika Anda ingin mengupload brosur Ertiga, rename file brosur asli menjadi `ertiga.pdf` dan timpa file yang sudah ada di folder ini.

## Catatan

- Format file harus PDF
- Pastikan nama file persis sama (case-sensitive)
- File akan langsung tersedia setelah diupload
- User dapat mendownload brosur dengan mengklik tombol "Download Brosur" di halaman detail produk
